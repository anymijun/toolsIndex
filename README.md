# 露营装备管理系统

一个基于 Vue 3 开发的 H5 露营装备管理应用，支持用户账号系统、装备包管理、装备检索等功能。

## 功能特性

### 👤 用户认证
- 手机号注册与登录
- 用户数据隔离存储
- 登录状态管理

### 🎒 装备包管理
- 创建装备包（支持图标选择）
- 删除装备包（含确认提示）
- 装备包列表展示

### 🔧 装备管理
- 添加装备（名称、类别、位置）
- 删除装备（含确认提示）
- 装备列表展示

### 🔍 装备检索
- 实时搜索装备
- 自动提示功能
- 显示装备所在位置

### 📦 装备类别
- **露营装备**：帐篷、炊具、照明、服装、鞋靴、背包等
- **车辆救援装备**：拖车绳、应急启动电源、充气泵、急救包等
- **车辆维修装备**：套筒扳手组、千斤顶、螺丝刀组、万用表等

## 技术栈

- **框架**: Vue 3 + Vite
- **路由**: Vue Router 4
- **状态管理**: localStorage
- **样式**: CSS3

## 快速开始


## 使用说明

### 测试账号

| 手机号 | 密码 | 昵称 |
|--------|------|------|
| 13800138000 | 123456 | 户外达人 |

### 注册账号

1. 点击"注册"切换到注册模式
2. 输入手机号码（11位）
3. 输入密码（至少6位）
4. 输入昵称（选填）
5. 点击"注册"完成

### 创建装备包

1. 点击右上角"+"按钮或底部"创建第一个装备包"按钮
2. 输入装备包名称
3. 选择图标
4. 点击"确认创建"

### 添加装备

1. 进入装备包详情
2. 点击"+ 添加装备"按钮
3. 输入装备名称
4. 选择装备类别
5. 输入存放位置
6. 点击"确认添加"

### 检索装备

1. 点击底部导航"检索"
2. 输入装备名称（支持自动提示）
3. 点击搜索结果查看位置

## 数据存储

- 用户信息存储在 localStorage 的 `camping_auth` 键中
- 装备数据存储在 localStorage 的 `camping_toolboxes` 键中
- 不同用户的数据通过用户 ID 隔离
